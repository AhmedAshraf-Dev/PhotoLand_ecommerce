# E-Commerce Platform

A modern e-commerce application built using React.js, Stripe, and Strapi, featuring dynamic product management, cart functionality, and responsive user experience.

## Features

- Product Listing
- Dynamic Product Details
- Shopping Cart Management
- Checkout Integration
- Responsive UI
- API Integration

## Technologies Used

- React.js
- Stripe
- Strapi
- JavaScript (ES6+)
- CSS / Tailwind CSS
